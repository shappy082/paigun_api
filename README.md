# How to use

1. clone git

```
git clone https://github.com/shappy082/paigun_api.git
```

2. Creat .env file at root dir

```
PORT = 3000
MONGODB_URI = mongodb://baac.topwork.asia:27017/alpha
DOMAIN = localhost
JWT_SECRET = baac@123
```

3. install

```
npm install
```

4. start

```
npm run dev
```
